# Real-Time Monitoring of Diver Status using Wireless Subsea Transmission (RWDM)

Real-Time Monitoring of Diver Status using Wireless Subsea Transmission (RWDM) is a bachelor project aimed at monitoring diver status in real time and communicating information through a wireless subsea transmission link.
This bachelor project marks the culmination of our studies in the TS3000 program at the Department of Microsystems, Faculty of Technology, Natural Sciences and Maritime Sciences. The work was carried out during the spring semester of 2026 and represents a collaborative research and development effort focused on Real-Time Wireless Diver Monitoring (RWDM).

## Repository overview

This repository contains components implemented in multiple languages:

- **C++ (56.3%)** — core/system components and performance-critical logic
- **TypeScript (17.8%)** — application/UI and supporting tooling
- **Python (15.6%)** — analysis, scripting, prototyping, or supporting services
- **C (2.3%)** — low-level/embedded or interoperability components
- **HTML (3.5%) / CSS (4.5%)** — web UI assets

## What this project is for

Typical goals for a system like this include:

- Collecting diver telemetry/status signals
- Transmitting data over a subsea wireless link
- Displaying/monitoring status in (near) real time
- Logging data for later review and analysis
